群组管理

按钮

1

点击跳转

群操作：

1、发送消息 2、添加，邀请成员【全员通知】3、设置/取消管理员【操作和被操作双方都没有通知修改为您已成为管理员，你已不是管理员；操作方无需通知】 4、主动退出群聊【全部人接收，修改为管理员接收；管理员主动退群和普通成员退群一致】4.1、群主退群：%%%/您 成为新群主—全员接收 

4.2【全员通知】群主解散：您/%%%已将群解散 5、编辑群头像【全部人接收，您/x x x修改了群组头像】 6、编辑群名称【您/x x xx更改群名称为 x x x x；自己那边有个 小bu g，聊天首页列表和里面会话提示不一致，前端看到改一下】7、创建群 【您/x x x创建了群 “x x x x”】8、移除群员：【全员通知：您/xx x移除了x x x x】

1、【身份规则】：

群身份：群主「最高级管理员」，管理员，普通成员

a）群主权限：解散/退出群聊；发消息；添加/邀请链接成员；移除成员/管理员；编辑群【默认开】；设置管理员；移除管理员

b）管理员权限：退出群聊；发消息；添加/邀请链接成员；移除成员；编辑群信息；不可以设置/移除管理员/群主

c）成员权限：退出群聊；发消息；添加成员；编辑群信息；

 基于权限打开的前提下

群操作 名字和操作名 之间要加个空格；

举例：您空格创建了群空格 “wahahhaha”

2

退出群组

按钮

3

点击跳转

1、【规则】无论是群主，管理员，群员退出群组，都保留他方的聊天记录，

若☑️清空聊天记录「默认不选中」，则直接从聊天列表清空该记录

2、 群主退出群组，群主权限自动给下一个成员「管理员优先」

有管理 => 随机选管理

无管理 => 随机选成员

3、若退出至最后一位，则该群解散

4、退出之后，用户无法接收和发送新消息

5、文案【您不再是群组成员，因此无法在群组传送讯息】

3

解散群组

按钮

3

点击跳转

1、群主【只有群主有】解散群组时，若☑️清空聊天记录「默认不选中」，则直接从聊天列表清空该记录，其他方的聊天记录保留。

2、解散之后显示文案【该群聊已被解散，因此无法在群组传送讯息】

3、群操作消息：您/%%%已将群解散

暂时无法选择把群主给谁【待优化】

4

设置/移除管理员

若身份是群主，则可以设置/移除群组管理员，成为管理员后，页面添加“使用链接加入群组”

5

从群组中移除

1、管理员/群主可以对一般成员进行移除操作，管理员可以对一般成员进行操作

2、移除之后【您不再是群组成员，因此无法在群组传送讯息】
